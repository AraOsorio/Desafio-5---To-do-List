# 📝 Todo List - Desafío 5

Aplicación web desarrollada para el Desafío 5 de Desafío Latam.  
Permite gestionar tareas dinámicamente utilizando JavaScript y manipulación del DOM.

---

## 🚀 Funcionalidades

- Agregar nuevas tareas
- Eliminar tareas
- Marcar tareas como completadas
- Contador automático de tareas totales
- Contador automático de tareas realizadas
- IDs correlativos que se reordenan al eliminar

---

## 🧠 Estructura de datos

Las tareas se almacenan en un arreglo de objetos:

```js
{
  id: number,
  descripcion: string,
  completado: boolean
}
```

---

## 🛠 Tecnologías utilizadas

- HTML
- CSS
- JavaScript (ES6)
- Manipulación del DOM
- Métodos de arreglos (`forEach`, `filter`, `findIndex`)

---

## ▶️ Uso

1. Abrir `index.html` en el navegador.
2. Escribir una tarea en el input.
3. Presionar **Agregar**.
4. Marcar como completada o eliminar según sea necesario.
